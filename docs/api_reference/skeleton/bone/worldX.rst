===================================
bone.worldX
===================================

| **Type:** ``number`` (read-only)
| **See also:** :doc:`index`

Overview:
.........
--------

The **world X position** of the bone. This is the bone's X position relative to the skeleton's origin.

Example:
--------
--------

.. code-block:: lua

   local bone = hero.bones[2]
   print("World X position:", bone.worldX)
